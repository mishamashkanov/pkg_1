# Конвертер цветовых моделей CMYK ↔ HLS ↔ XYZ

## Описание
Десктоп-приложение на Java для интерактивного преобразования цветов между промышленными цветовыми моделями, используемыми в полиграфии, дизайне и компьютерной графике. Приложение позволяет работать с моделями CMYK (полиграфия), HLS (компьютерная графика) и XYZ (аппаратно-независимая модель), изменять значения компонентов с помощью ползунков и числовых полей, а также визуально наблюдать результат на цветовой панели.

## Возможности

**Поддержка трёх профессиональных цветовых моделей:**
- **CMYK** - субтрактивная модель для полиграфии (Cyan, Magenta, Yellow, Key/Black)
- **HLS** - интуитивная модель для работы с оттенками (Hue, Lightness, Saturation)  
- **XYZ** - аппаратно-независимая модель CIE 1931, основа цветовых преобразований

**Интерактивное управление цветом:**
- Двусторонняя синхронизация значений между моделями
- Три режима ввода: ползунки, числовые поля, визуальный выбор из палитры
- Визуальная панель предпросмотра текущего цвета
- Отображение HEX-значения цвета

**Профессиональные функции:**
- Контроль цветового охвата с предупреждениями
- Автоматическая коррекция выходящих за диапазон значений
- Точные математические преобразования между моделями
- Красивый и интуитивно понятный интерфейс

## Математический аппарат

### Преобразования CMYK ↔ RGB
```java
R = 255 × (1 - C) × (1 - K)
G = 255 × (1 - M) × (1 - K)  
B = 255 × (1 - Y) × (1 - K)
```

### Преобразования HLS ↔ RGB
```java
// Цилиндрические координаты в RGB пространстве
H = hue ∈ [0°, 360°]
L = lightness ∈ [0%, 100%]
S = saturation ∈ [0%, 100%]
```

### Преобразования XYZ ↔ RGB
```java
// Матрица преобразования XYZ to RGB (sRGB)
R = 3.2406·X - 1.5372·Y - 0.4986·Z
G = -0.9689·X + 1.8758·Y + 0.0415·Z
B = 0.0557·X - 0.2040·Y + 1.0570·Z

// Гамма-коррекция sRGB
linearToSrgb(u) = (u ≤ 0.0031308) ? 12.92·u : 1.055·u^(1/2.4) - 0.055
```

## Как это работает

1. **Выбор исходной модели**  
   Пользователь изменяет значения компонентов в одной из моделей — CMYK, HLS или XYZ

2. **Каскадная конвертация**  
   Приложение выполняет точные математические преобразования между моделями:
   - CMYK ↔ RGB (полиграфические преобразования)
   - RGB ↔ HLS (цилиндрические координаты)  
   - RGB ↔ XYZ (матричные преобразования)

3. **Визуализация и контроль**
   - Цвет отображается в блоке предпросмотра
   - Показывается HEX-представление цвета
   - Система предупреждает о некорректных значениях
   - Все модели автоматически синхронизируются

4. **Работа с палитрой**  
   Встроенный цветовой выбор позволяет быстро выбирать оттенки и анализировать их представление в разных моделях

## Установка и запуск

### Требования
- Java Runtime Environment (JRE) 8 или выше
- Windows/Linux/macOS

### Запуск приложения

**Способ 1: Готовый JAR (рекомендуется)**
```bash
java -jar ColorConverter.jar
```

**Способ 2: Сборка из исходников**
```bash
# Скомпилировать проект
build.bat

# Запустить приложение  
run.bat
```

## Пример использования

1. **Открыть приложение**
2. **Исследовать цвет #FF0000 (красный):**
   - В CMYK: C=0%, M=100%, Y=100%, K=0%
   - В HLS: H=0°, L=50%, S=100% 
   - В XYZ: X=41.24, Y=21.26, Z=1.93

3. **Изменить компоненты:**
   - Установить C=100% в CMYK и наблюдать автоматическое обновление HLS и XYZ
   - Изменить Hue в HLS на 120° (зеленый) и проанализировать изменения в CMYK

4. **Использовать цветовую палитру:**
   - Выбрать любой цвет из диалога выбора
   - Проанализировать его представление во всех моделях
